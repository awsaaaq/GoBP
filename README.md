# GoBP
An Bypass Antivirus Tool by Golang in windows


 - Bypass Windows Defender
 - Bypass Kaspersky
 - Bypass 360 
 - ...
# Usages
`
git clone https://github.com/awsaaaq/GoBP.git
`


Put payload.bin in main.go's directory



`
go build main.go
`

Execute main.exe

Generate an execute GoBP.exe
