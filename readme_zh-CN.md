# GoBP
免杀工具 Go语言编写
 - 免杀生成工具
 - 语言：Golang
 - 平台：Windows
 - 目前免杀杀软
   - Windows defender
   - 卡巴斯基


# 使用方法

`
git clone https://github.com/awsaaaq/GoBP.git
`

将payload.bin 放置在和 main.go同目录文件夹中

`
go build main.go
`

执行main.exe
![image](images/mainexe.png)

生成一个GoBP.exe
Generate an execute GoBP.exe

![image](/images/payload.png)

# 注意事项
在执行main.exe前关闭杀软！

关闭杀软自动上传功能！
